The Science

From Maxwell's equations to a predictive instability boundary — how physics tells us when a wireless link is about to fail.

Foundation

Where it starts: electromagnetic wave propagation

Maxwell's Equations

All wireless communication is governed by Maxwell's equations. A time-varying electric field creates a magnetic field, and vice versa — producing electromagnetic waves that carry data through space.

∇ × E = −∂B/∂t

Faraday's law: a changing magnetic field induces an electric field.

What happens in practice

In real environments, signals reflect off walls, scatter off objects, and diffract around edges. The receiver sees a superposition of many copies of the original signal, each delayed and phase-shifted.

This creates a multipath channel — complex, time-varying, and frequency-dependent.

Channel Model

The wireless channel as a matrix

In a MIMO system with multiple antennas, the channel between transmitter and receiver is described by a matrix H. Each element Hij captures the complex gain from transmit antenna j to receive antenna i.

y = Hx + n

y = received signal, x = transmitted signal, n = noise.

The base station periodically estimates H — this estimate is called Channel State Information (CSI). But by the time the CSI is used for beamforming, it may already be outdated.

Key parameters

  • τf — CSI feedback delay (how old the estimate is)
  • Tc — Channel coherence time (how long the channel stays the same)
  • fD — Doppler frequency shift (proportional to user speed and carrier frequency)
  • κ(H) — Condition number of H (how sensitive it is to perturbation)
The Decay

How CSI becomes stale

The correlation between the true channel and the estimated channel decays with time. Using the Jakes autocorrelation model:

ρ(τf) = J₀(2π · fD · τf)

where J₀ is the zeroth-order Bessel function and fD = v·fc/c is the maximum Doppler frequency.

τf ≪ Tc
CSI is fresh → ZF works
τf ≈ Tc
CSI is aging → instability zone
τf > Tc
CSI is stale → ZF collapse
The Amplifier

Condition number makes it worse

The condition number κ(H) measures how sensitive the channel matrix is to small errors. When κ is large, even tiny CSI errors blow up when inverted.

‖ΔW‖/‖W‖ ≤ κ(H) · ‖ΔH‖/‖H‖

The relative error in beamforming weights W is bounded by κ times the relative channel error.

This is why the instability is nonlinear: it's not just that CSI ages — the conditioning of the channel amplifies that aging.

Core Metric

The Conditioned Coherence Index

We combine CSI age and condition number into a single predictive metric.

CCI = κ(H) · (τf / Tc)
CCI < 0.5
Stable region. ZF works as intended.
0.5 ≤ CCI < 1
Transition zone. Performance is degrading.
CCI ≥ 1
Coherence cliff. Switch to MRT immediately.
This is the key insight: the coherence cliff isn't a gradual decline — it's a sharp phase transition. CCI captures both the rate of CSI aging and how badly the channel amplifies that aging.
Beamforming Modes

ZF vs. MRT

Zero-Forcing (ZF)

Cancels multi-user interference by inverting the channel matrix. Requires accurate, up-to-date CSI.

Property Value
Multiplexing Full spatial multiplexing
Interference Cancelled (when CSI is fresh)
Sensitivity to CSI age Very high
Failure mode Sudden collapse (coherence cliff)

Maximum Ratio Transmission (MRT)

Maximizes signal strength at each user without trying to cancel interference. More robust to stale CSI.

Property Value
Multiplexing Reduced (no interference control)
Interference Present but bounded
Sensitivity to CSI age Low
Failure mode Gradual degradation
Mathematical Model

The instability derivation

How we go from physics to a computable risk metric.

Step 1: Channel autocorrelation decay

Using the Jakes model, channel correlation decays as:

ρ(τ) = J₀(2π·fD·τ)

For small arguments, this approximates as ρ ≈ 1 − (π·fD·τ)². The key observation is that τf/Tc is a dimensionless ratio that controls when this decay becomes significant.

Step 2: ZF weight error bound

ZF weights are computed as W = H† (pseudo-inverse). The standard perturbation bound gives:

‖ΔW‖/‖W‖ ≤ κ(H) · ‖ΔH‖/‖H‖ / (1 − κ(H)·‖ΔH‖/‖H‖)

This means the beamforming error grows super-linearly with κ. When κ·‖ΔH‖/‖H‖ approaches 1, the bound diverges — this is the coherence cliff.

Step 3: Combining into CCI

The CSI error ‖ΔH‖/‖H‖ is proportional to 1−ρ(τf), which scales as (τf/Tc)². Multiplying by κ(H) gives the CCI:

CCI = κ(H) · τf / Tc

CCI ≈ 1 marks the transition boundary. Below it, ZF is optimal. Above it, ZF is destructive and MRT provides better performance.

Step 4: Risk metric and switching rule

We define a risk metric:

Risk = 1 − e−CCI²

This maps CCI onto [0, 1] with a sharp transition near CCI = 1. The switching rule is simple: when Risk exceeds a threshold (default 0.3), switch from ZF to MRT. When Risk drops back below the threshold, switch back.

Step 5: Probability of collapse

Using the exponential model for error growth, the probability that ZF will produce catastrophic interference is:

Pcollapse = 1 − e−(CCI/CCIcrit)²

where CCIcrit ≈ 1 is the critical value. This gives a smooth but steep transition — small changes in CCI near the cliff cause large changes in collapse probability.
